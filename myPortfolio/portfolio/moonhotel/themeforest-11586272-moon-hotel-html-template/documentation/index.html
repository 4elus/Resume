Moon - Hotel HTML5 Template

  • Created: 26/May/2015
  • Latest update: 29/May/2015
  • By: Theme-Paradise
    • support@theme-paradise.com

Moon - Hotel HTML Template is unique and beautifully crafted for hotel services. It is responsive, based on Bootstrap 3+ and W3C valid. The HTML template has 15 Pages including 2 home pages. The template is full and easily customizable. The Design of HTML template is based on "Moon - Hotel PSD Template" PSD design.

Contents

You should find the following files and folders in the download pack:
  • bootstrap/
  • css/
  • images/
  • js/
  • less/

bootstrap/ - contains bootstrap v3.3.4 version. It is unmodified, and you can replace it with any version as long as compatibility with our template is maintained.
css/ - LESS-compiled CSS files plus some small JS Plugin related files are here.
fonts/ - The fonts used in our templates are here. Currently FontAwesome and is in this folder.
images/ - images in our templates are here.
js/ - Javascript files are in this folder.
less/ - We created stylesheet codes in LESS format for better maintenance and future changes, and you can find all LESS files here.
index1.html - Home page with default header.
index2.html - Home page with dark header and other contents.
about.html - About us page with testimonials and accordion examples.
blog1.html - Blog version 1 with posts in image mosaic layout.
blog2.html - Blog version 2 with posts in list layout and a right sidebar.
blog-single.html - Single post page with comments and social sharing icons
booking-choose-date.html - Booking page with date selection
booking-choose-room.html - Booking page with room selection.
booking-reservation.html - Booking page with reservation features.
contact.html - Contact us page with google map and contact us form.
facilities.html - Shows facilities in grid format.
gallery.html - List of galleries .
gallery-single.html - Single gallery page with image lightbox2.
rooms.html - Page that shows different rooms
room-single.html - Room page with detailed description
